Class:
Usage: 

<button class="btn no-select effect #33e-bg">Button</button>
			


			

btn

General button class, required.
On clic event: brightness applied.


no-select

Disable user's selection, optional.


effect

Effect (brightness) applied on over/clic, optional.


#33e-bg

Background color in Hex code (always 3 digits) followed by -bg suffix, optional.
See the Custom CSS properties (CSS variables) where --bg-action defines this color.


<a href="#!" class="btn no-select #33e-bg">Link button</a>
			

Link button
			

<input type="submit" value="Input button" class="btn #33e-bg" />
			


			

<input type="reset" value="Reset button" class="btn outline #fff-bg" />
			


			

outline

Creates outline element, optional.


<label role="button" class="btn no-select #33e-bg">Label button</label>
			


			


<a href="#!" class="btn trans no-select">Transparent Link</a>
			

Transparent Link
			

trans

Creates transparent element, optional.


<button class="btn no-select effect ripple #33e-bg">Button ripple</button>
			


			

ripple

Adds a Google Material ripple effect on click event (effect class required), optional.
